CareApp 개인정보처리방침

최종 업데이트: 2026-06-28 · 앱: CareApp (육아 기록 다이어리)

CareApp은 이메일·이름·전화번호 등 개인을 식별할 수 있는 정보를 일절 수집하지 않습니다. 회원가입 없이 사용하며, 계정은 사용자의 기기에서 생성된 보안 키로만 보호됩니다.

1. 수집·생성하는 정보

  • 기기 계정 식별자: 앱 최초 실행 시 기기에서 공개키/비공개키 쌍을 생성합니다. 서버에는 임의로 발급된 계정 ID와 공개키만 저장되며, 비공개키는 기기의 보안 저장소(Android Keystore / iOS Keychain)에만 보관되어 외부로 전송되지 않습니다.
  • 육아 기록 데이터: 사용자가 입력한 수유·수면·배변·체온·투약·성장 측정·예방접종·발달 기록. 기본적으로 사용자 기기에 저장됩니다.
  • 표시 이름(선택): 가족 공유 시 구성원 구분용으로 사용자가 직접 입력할 수 있는 별명입니다. 입력하지 않아도 됩니다.
  • 이메일, 이름, 전화번호, 위치정보, 광고 식별자, 연락처 등은 수집하지 않습니다.

2. 정보의 이용 목적

  • 육아 기록의 저장·표시 및 통계·성장 곡선 등 분석 제공
  • 가족 구성원 간 기록 공유 및 기기 간 동기화(사용자가 가족 그룹을 사용하는 경우에 한함)
  • 기기 키 기반의 안전한 인증

3. 저장 및 공유

  • 기록 데이터는 사용자의 기기에 저장됩니다.
  • 사용자가 가족 공유 기능을 사용할 때에만, 해당 가족 그룹 단위로 기록이 서버(api.wanirosa.com)에 동기화되어 같은 그룹의 구성원이 함께 볼 수 있습니다.
  • 수집한 정보를 제3자에게 판매하거나 광고 목적으로 제공하지 않습니다.

4. 기기 이전 및 복구

사용자가 복구 암호를 설정하면, 계정 복구에 필요한 비밀값을 복구 암호로 암호화한 형태로만 서버에 보관합니다. 서버는 복구 암호 평문이나 비공개키를 알지 못합니다. 새 기기에서 계정 ID와 복구 암호로 계정을 안전하게 이전할 수 있으며, 이전 시 기존 기기의 접근 권한은 폐기됩니다.

5. 외부 데이터

  • 예방접종 안내는 대한민국 질병관리청의 공개 정보를 사용합니다.
  • 성장 표준 곡선은 세계보건기구(WHO) 및 2017 소아청소년 성장도표 기준의 공개 통계를 사용합니다.
  • 이들 조회 시 개인 식별정보는 전송되지 않습니다.

6. 보안

비공개 키는 기기의 하드웨어 보안 저장소에 보관되고, 서버 통신은 HTTPS로 암호화됩니다. 복구용 비밀값은 사용자만 아는 복구 암호로 암호화되어 보관됩니다.

7. 데이터 삭제

앱 내 설정의 “데이터 초기화”로 기기의 기록을 삭제할 수 있습니다. 가족 서버에 동기화된 데이터의 삭제를 원하시는 경우 아래 연락처로 요청해 주세요.

8. 아동의 개인정보

본 앱은 보호자(부모 등)가 자녀의 양육 정보를 기록하기 위한 도구이며, 아동으로부터 직접 정보를 수집하지 않습니다. 입력되는 자녀 관련 정보는 보호자가 자신의 기기에서 관리합니다.
